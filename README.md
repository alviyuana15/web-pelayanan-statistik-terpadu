# web-pelayanan-statistik-terpadu
Web-based integrated statistical service system developed during internship at BPS. Provides data access, request management, and service information.
Website ini dikembangkan selama magang di Badan Pusat Statistik (BPS).

## Fitur
- Akses data statistik
- Permintaan layanan data
- Informasi publik statistik
- Interface user-friendly
